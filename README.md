# 🚀 DevUp - Landing Page

Landing page oficial da **DevUp**, uma startup especializada na criação de sites institucionais modernos, rápidos e focados em conversão.

---

## 🎯 Sobre o projeto

Este repositório contém uma landing page desenvolvida em um único arquivo HTML, com foco em simplicidade, performance e fácil deploy.

---

## 💡 Proposta da DevUp

Oferecemos um modelo direto ao ponto:

- ✔ Site institucional completo
- ✔ Design moderno e responsivo
- ✔ Integração com WhatsApp
- ✔ Foco total em conversão

> Sem complicação. Sem planos confusos.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (interno)
- JavaScript (interno)

---

## 📱 Funcionalidades

- Layout responsivo (mobile-first)
- Botão de contato via WhatsApp
- Estrutura otimizada para conversão
- Carregamento rápido (sem dependências externas pesadas)

## 🚀 Como rodar o projeto

```bash
# Clone o repositório

# Entre na pasta
```
